# Sapbaq Admin — سَبّاق الإدارة (Admin & Driver App)

The management and delivery companion app for the **Sapbaq (سَبّاق)** ecosystem:
admins assign orders to drivers and workshops; drivers see their delivery
destinations and upload delivery proof (photos/videos).

Package: `sapbaq_admin` · Application ID: `com.albairakgroup.saqia.admin`

## Getting Started

```sh
flutter pub get
flutter gen-l10n
dart run flutter_launcher_icons   # regenerate launcher icons after changing assets/icon/
flutter run
```
